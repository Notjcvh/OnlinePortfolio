My Contributions

ODYSEED is a project that was created for my GAME490 class at George Mason University. ODYSEED is an adventure style game where the player has to go on a quest in order to save his island. On this project I was in charge of game programming. I played a central role in bug fixes, enemy and boss AI, handling the git, abilities, balancing, and UI programming. If you're interested in learning more click here for a more in depth presentaton.
